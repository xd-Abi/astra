# System Requirements

To install the browser, you need to ensure that your system meets the following requirements:

| Operating System | Processor          | Memory | Storage |
| ----------------- | ------------------ | ------ | ------- |
| Windows 7/8/8.1/10 64-bit | 64-bit 2.0 GHz | 2 GB   | 500 MB  |
| Ubuntu 16.10 & Newer Versions | 64-bit 2.0 GHz | 2 GB   | 500 MB  |
| Mac OS X & Newer Versions | 64-bit 2.0 GHz | 2 GB   | 500 MB  |
