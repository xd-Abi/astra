# Installation

1. Clone the repository using `git clone --recursive https://github.com/xd-Abi/astra`.
2. To start the application, navigate to the desired folders and install the packages.

### Install the Search Engine

```sh
cd astra/web
yarn && yarn dev
```

### Run the Browser

1. Open the project with CLion.
2. Allow the `astr/browser/source/main.cpp` to run.
